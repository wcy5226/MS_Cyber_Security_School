<깃허브> 설정하기

-github 가입 : https://github.com/
New repository		(리포지터리 생성, 설정에서 .gitignore 선택)
<>Code 클릭 후 https 복사

-git 설치 : https://git-scm.com/

-cmd 실행

git config --global user.email "github 가입한 이메일"
git config --global user.name "github 가입한 이름"
git clone [https://github.com/사용자명/리포지터리이름.git] [복사할 폴더명]		(git 리포지터리 > 폴더로 복사)



-받은 파일을 업로드할 폴더로 이동
-notepad up.bat		(up.bat파일 생성하고 메모장으로 열기)
-아래 내용 메모장에 입력(up 실행하면 cmd에서 입력된 내용 실행함)

git add . / git add *	(git add . : 변경된 파일 추가<<일반적으로 사용/ git add * : 변경된 숨김 파일까지 추가 )
git commit -m [메세지] 	(git add로 저장한 변경사항을 로컬 저장소에 저장 / -a 코멘트 없이 올림 / -m [메시지] 코멘트 업로드)
git push			(로컬 저장소(.git 폴더)에 저장된 내용을 리포지터리에 업로드)
